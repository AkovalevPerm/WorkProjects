Название xml-файла должно совпадать с название типа сообщения в xml-модели сообщений (константа NameForESB), иначе будут падать тесты ServiceBusListner.Tests Synchronization.Tests
Xml-файлы с окончанием Template представляют собой шаблон, с которым сравнивается сгенерированная одноименная xml без Template.
Xml-файлы файлы должны быть помечены атрибутом обязательного копирования в Output Directory.
